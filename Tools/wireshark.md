# Wireshark

## Overview
- Wireshark is an industry-standard tool used to capture and analyse network traffic.
- It focuses on packet data (PCAP files) to investigate network activity.

## Key Concepts
- Analyses network traffic for troubleshooting and detecting suspicious activity.
- Uses panels such as packet list, packet details, and packet bytes.
- Packet dissection breaks down packets into protocol layers.
- Packet navigation helps locate and analyse specific packets.
- Packet filtering isolates relevant traffic for analysis.

## Relevance
- Crucial for troubleshooting network issues.
- Helps detect malicious or unusual traffic.
- Used for analysing and debugging network communications.

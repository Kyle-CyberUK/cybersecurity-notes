# PowerShell

## Overview
- PowerShell is a powerful command-line shell and scripting language used for task automation and configuration management in Windows environments.
- PowerShell is object-oriented, allowing users to work with structured data instead of plain text.

## Cmdlets and Help System
- PowerShell uses cmdlets instead of traditional commands.
- Cmdlets follow a verb-noun format (e.g. Get-Help).
- Get-Command - Lists available commands.
- Get-Help - Explains how to use a command.

## Navigating Files
- PowerShell manages the filesystem similarly to command prompt but with more flexibility.

## Piping
- The pipe operator (|) sends output from one command directly to another.
- Powerful for data processing and analysis.

## Real-Time System Monitoring
- PowerShell monitors system activity in real time.

## Scripting and Automation
- PowerShell supports scripts (.ps1 files).
- Tasks such as user management can be automated to save time and reduce manual work.

## Relevance
- PowerShell is relevant for system administration, penetration testing, and incident response.

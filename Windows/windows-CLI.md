# Windows CLI

## Overview
- The Command Prompt is a text-based interface used for interacting with the Windows operating system.
- It is used because it provides faster navigation and administration.

## Terminal Navigation
- cd - Displays your current location.
- cd <foldername> - Change directory.
- dir - Lists files and folders in current directory.
- dir /a - Lists all folders and files in current directory.
- dir /s - Search subdirectories.
- cd .. - Go back one directory.
- cd \ - Go to root directory.
- type <filename.txt> - Displays contents of the file.

## System Information
- whoami - Displays currently logged-in user.
- hostname - Displays system hostname.
- systeminfo - Basic system info including OS version, installed patches, and memory info.
- ipconfig - Shows system network configuration.

## Relevance
- Knowledge of Windows CLI is essential for faster workflow, administrative tasks, remote management, and scripting and automation.

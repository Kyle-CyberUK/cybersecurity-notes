# JavaScript Basics

## Overview
- JavaScript is a programming language commonly used to make websites interactive.

## Variables and Constants
- A variable is a named container that can change as the program runs. Declared with the 'let' keyword.
- E.g. let guess = 0;
- A constant is a named container that does not change once set. Declared with the 'const' keyword.
- E.g const secret = 10;

## Data Types
- Data types define what kind of value a variable can store.
- They can be either primitive (basic) types, or reference (complex) types.
- Primitive types store single simple values, such as strings, numbers, booleans, etc.
- Reference types store collections or complex structures, such as functions or arrays (lists).

## Conditionals
- Conditionals allow a program to make decisions based on specific conditions.
- Uses if, else if, and else statements.
- Comparison operators compare values inside conditional statements.
- !== means "not equal to" and compares both value and data type.
- === means "equal to" and also compares both value and data type.

## Loops
- Loops let the program repeat a set of instructions until a condition changes.
- Loops prevent the developer from repeating the same code.

## Relevance
- Understanding how JavaScript works helps identify and prevent client-side vulnerabilities.

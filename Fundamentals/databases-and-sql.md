# Databases and SQL

## Overview
- A database is a structured system for storing and organizing information, via the use of tables, so it can be efficiently retrieved and managed.

## Tables, Rows, and Columns
- Tables - Hold collections of related data.
- Rows - Represent individual records.
- Columns - Define the type of data each part of a row contains.

## SQL Basic Queries
- SELECT - Retrieve data (use * to select all columns).
- FROM - Specify which table.
- WHERE - Filter results.
- ORDER BY - Sort data (use DESC for descending order).

## Relevance
- Learning databases and SQL is essential for data management, data analysis, cybersecurity awareness, and backend development.

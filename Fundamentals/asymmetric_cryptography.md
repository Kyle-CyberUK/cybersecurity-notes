# Asymmetric Cryptography

## Overview
- Asymmetric cryptography uses key pairs to enable secure communication.
- It is widely used in modern security protocols to protect data over networks.

## Key Concepts
- Uses a **public key** and **private key**.
- Data encrypted with one key can be decrypted with the other.
- Used in protocols such as HTTPS and SSH.
- **Rivest-Shamir-Adleman (RSA)** is a common algorithm based on factoring large numbers.
- **Diffie-Hellman** enables secure key exchange over insecure networks.
- **Digital signatures** verify authenticity and integrity.
- **Certificates** bind public keys to identities.

## Relevance
- Essential for understanding secure communication and encryption.
- Forms the foundation of HTTPS, SSH, and other secure protocols.
- Protects data confidentiality, integrity, and authenticity.

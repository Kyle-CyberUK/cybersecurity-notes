# Operating Systems

## Overview
- An operating system is core system software that manages a computer's hardware, software, and resources.

## System Privilege Layers
- Kernel space - Highly privileged area, interacts directly with the hardware.
- User space - Regular apps run with limited access protecting system stability.

## Responsibilities of an OS
- Process Management - Starts, schedules, and stops programs.
- Memory Management - Allocates RAM to programs, uses virtual memory when needed.
- File System Management - Organises files and folders, manages permissions.
- User Management - Manages login, accounts, and access controls.
- Device Management - Loads drivers and abstracts hardware for programs.

## OS Security
- Authentication - Verifies your identity.
- Permissions - Controls what users and programs can read, write, and execute.
- Isolation - Each process runs in its own protected space.
- System protection - Protects system files, prevents unauthorized system changes, maintains stability.

## User Interfaces
- GUI (Graphical User Interface) - Visual interaction (windows, menus)
- CLI (Command Line Interface) - Text interface for typing commands.

## Real World OS's
- Desktop - Windows, Linux, MacOS.
- Server - Linux server, Windows server, Unix variants.
- Mobile - Android, iOS
- Embedded/IoT - Linux-based for specialized devices or real-time OS when guaranteed response times are required.

## Relevance
- Essential for understanding cybersecurity, networking, and system administration.

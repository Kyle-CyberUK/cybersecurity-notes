# Computer Components

## Overview
- Core hardware required for a computer to function.

## Key Components
- Motherboard - Connects all components.
- CPU (Central Processing Unit) - Continuously executes instruction.
- RAM (Random Access Memory) - Keeps relevant information temporarily for the CPU to access.
- Storage devices (HDD/SSD) - Permanent data storage.
- PSU (Power Supply Unit) - Supplies energy to all components.
- GPU (Graphics Processing Unit) - Receives graphical information and displays it.
- Network adapter - Allows the computer to communicate with other systems.

## Relevance
- Important for diagnosing hardware issues in an IT support role.

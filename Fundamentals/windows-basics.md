# Windows Basics

## Overview
- Provides a graphical interface to manage hardware, software, files, users, and system settings.

## Account Types
- Guest - Restricted, allows for temporary access (often disabled by default).
- Standard - Used for normal use.
- Administrator - Allows full control.

## Windows Desktop
- Desktop - Workspace for files, folders, and shortcuts.
- Start menu & Search - Open apps and access power options quickly.
- Task view - Displays open windows.
- Network & Audio settings - Quickly access network/audio settings.
- Date & Time - Shows date and time, as well as opening a full calendar.
- Notifications - Displays computer and application notifications.

## File Management
- Windows uses hierarchical folder structure.
- File Explorer manages folders and files.

## System Config Tools
- Windows Settings - Modern configuration for managing system, devices, accounts, and updates.
- Control Panel - Legacy tool built for older configuration options.

## Task Manager 
used to monitor system activity.
- Processes - Shows running apps & resource usage.
- Performance - Displays CPU, RAM, storage, Wi-Fi, and GPU graphs.
- Users - Shows logged in users.
- Details & Services - Technical view of processes and services.

## Windows Security
- Virus & threat protection - Scans for malware.
- Firewall & network protection - Control network traffic.
- App & browser control - Protects from malicious apps & browsers.
- Device Security - Hardware-based protection.

## Windows Firewall
- Controls inbound/outbound traffic.
- Contains 3 profiles; Domain, Private, and Public.
- Advanced settings allow creating rules.

## Relevance
- Provides foundational knowledge for managing and securing Windows environments.

# Linux CLI

## Overview
- CLI (Command Line Interface) is a text-based way to interact with the operating system.
- The home directory is represented by the symbol: ~

## Terminal Navigation
- pwd - Print working directory. Displays current folder.
- ls - List files/folders in a directory.
- ls -l - Provides more details.
- ls -a - Shows hidden files (files beginning with a dot).
- cd <directory> - Change directory.
- cd .. - Moves up one directory level.
- find <path> -name <filename> - Search for a certain file.
- cat <file> - Displays file contents.

## System Information
- whoami - Displays currently logged-in user.
- uname -a - Displays kernel and OS version info.
- df -h - Human-readable disk usage info.
- hostname - Displays system hostname.

## Relevance
- Essential for working with Linux servers and cybersecurity tools.
- Required knowledge for penetration testing, system administration, and cloud environments.

# Defensive Security

## Overview
- Defensive security involves protecting systems, detecting attacks, and responding to security incidents.

## Blue Teaming
- Focuses on 2 main goals:
  - Preventing attacks before they occur.
  - Detecting and responding to attacks as they occur.

## Understanding the Environment
- This includes identifying:
  - Critical systems and devices.
  - How systems communicate.
  - Which assets are top priority.

## Defending the Environment
- Defenders apply security controls such as:
  - Firewalls and Intrusion Prevention Systems (IPS).
  - Security monitoring and log analysis.
  - Regular system updates.
  - Access controls and security policies.
- Risk prioritization involves focusing security efforts on the most critical systems first.

## Relevance
- Defensive security helps organisations detect threats early, prevent attacks, and minimise damage.

# Data Representation

## Overview
- Computers represent numbers, colours, and other data using bits,

## Representing Colours
- Computers use Red, Green, and Blue to create colours.
- Each colour can be On (1) or Off (0).
- 3 bits = 8 colours.
- Each RGB value uses 8 bits (1 byte).
- 8 bits = 256 possible values = 16,777,216 colours.
- A full colour uses 24 bits (3 bytes).

## Hexadecimals
- 4 bits = 1 hexadecimal digit (0-9, A-F).
- Example: A7EBF5 represents Cyan.
- A7 = Red, EB = Green, F5 = Blue.

## Number Systems
- Decimal (Base-10) – Humans use digits 0–9.
- Binary (Base-2) – Computers use 0 and 1.
- Hexadecimal (Base-16) – Convenient representation of binary (0–9, A–F).
- Octal (Base-8) – Short form of binary (0–7).

## Relevance
- Understanding how data is encoded is essential for programming, graphics, and low-level system operations.

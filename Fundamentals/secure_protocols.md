# Secure Protocols

## Overview
- Secure protocols protect data as it travels across networks using encryption and authentication.
- TLS is a cryptographic protocol, it secures communications over a network.

## Key Concepts
- Uses TLS to secure data in transit, providing confidentiality, integrity, and authenticity.
- HTTPS uses TLS to encrypt communication, protecting confidential information.
- SSH provides encrypted remote access to systems.
- SFTP uses SSH, while FTPS uses TLS for encryption.
- SMTPS encrypts email sending, while POP3S and IMAPS protect email retrieval.
- Certificates verify the identity of servers and establish trust.

## Relevance
- Secure protocols are critical because data is constantly transmitted across untrusted networks.
- Secure protocols help prevent attackers from intercepting and modifying data.

# HTTP Protocol

## Overview
- HTTP (Hypertext Transfer Protocol) is used to transfer web data between a client and server.

## Key Concepts
- Operates at the application layer.
- Uses TCP port 80.
- Each request is independent.
- Common methods include GET, POST, PUT, DELETE.

## HTTPS
- Secure version of HTTP.
- Uses TLS encryption, three way handshake.
- Runs on port 443.

## Relevance
- An understanding of HTTP helps diagnose web access and browser-related issues.

# Networking Essentials

## Overview
- Networking enables devices to connect, communicate, and exchange data efficiently.

## Network Configuration
- DHCP (Dynamic Host Configuration Protocol) automatically assigns devices:
  - IP address
  - Subnet mask
  - Default gateway
  - DNS server
- Uses the DORA process:
  - Discover – Device searches for a DHCP server.
  - Offer – Server offers an IP configuration.
  - Request – Device requests the offered IP configuration.
  - Acknowledge – Server confirms the assignment.

## IP and MAC addressing
- Devices use IP addresses to identify each other on a network.
- MAC addresses identify physical network interfaces.
- ARP (Address Resolution Protocol) maps an IP address to a MAC address so devices can communicate locally.

## Network Diagnostics
- ICMP (Internet Control Message Protocol) is used for troubleshooting.
- Common tools include:
  - ping - Checks if a device is reachable.
  - traceroute - Shows the path data takes across a network.

## NAT (Network Address Translation)
- Allows multiple devices on a private network to share a single public IP address.
- Translates internal IP addresses to a public IP for external communication.

## Relevance
- Networking is essential to help professionals:
  - Understand how data moves across systems.
  - Troubleshoot connectivity issues.
  - Detect suspicious activity.
  - Analyse traffic.
  - Perform network scanning.

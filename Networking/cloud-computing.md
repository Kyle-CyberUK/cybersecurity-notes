# Cloud Computing

## Overview
- Allows users to use computers, storage, and services over the internet.

## Cloud Benefits
- Scalability - Change resources as needed.
- On-Demand Self Service - Create/remove resources instantly.
- Pay-as-you-go - Only pay for the resources you use.
- Security & Reliability - Providers secure the infrastructure.
- Global access - Connect from anywhere.

## Cloud Types
- Public cloud - Shared, affordable, and easy to scale.
- Private cloud - Dedicated for one organisation. Offers greater control, customization, and compliance.
- Hybrid cloud - A combination of public and private.

## Cloud Service Models
- IaaS (Infrastructure as a Service) - Rent basic computing resources; you manage OS & apps.
- PaaS (Platform as a Service) - Provider manages infrastructure & OS; you focus on apps.
- SaaS (Software as a Service) - Provider manages everything; you use the app (e.g. Gmail).

## Major Cloud Vendors
- AWS (Amazon Web Services)
- Microsoft Azure
- Google Cloud
- Alibaba Cloud
- IBM Cloud
- Oracle Cloud

## Example Service
- EC2 (Elastic Compute Cloud) is an AWS web service that provides scalable virtual servers in the cloud.

## Relevance
- Widely used for web hosting, data storage, cybersecurity labs, and enterprise systems.

# DNS (Domain Name System)

## Overview
- DNS translates human-readable domain names into IP addresses.

## Key concepts
- Uses a hierarchical structure (root, TLD, authoritative servers)
- Commonly uses UDP port 53.
- DNS records are instructions telling the internet how to handle requests.

## Process
- Client sends DNS query.
- Resolver contacts the appropriate DNS servers.
- IP address is returned to client.

## Relevance
- Important for troubleshooting connectivity and website issues in IT support.
